## 🎯 วิธีการทำงาน

1. **ผู้ใช้กรอกฟอร์ม** → กรอกข้อมูล: ชื่อเด็ก, ช่วงอายุ, ชื่อผู้ปกครอง, เบอร์โทร
2. **กดส่งใบสมัคร** → ระบบจะ validate ข้อมูล
3. **ส่งไป Telegram** → ข้อมูลจะถูกส่งไปยัง Telegram Bot ทันที
4. **แสดงผลสำเร็จ** → ผู้ใช้เห็นข้อความว่าส่งสำเร็จ

## ⚙️ การตั้งค่า (ถ้าต้องการเปลี่ยน Bot)

ถ้าคุณต้องการใช้ Telegram Bot ของคุณเอง:

1. สร้าง Telegram Bot ใหม่ (ผ่าน @BotFather)
2. สร้าง Cloudflare Worker (หรือ backend อื่นๆ)
3. เปลี่ยน URL ใน `AcademyPage.jsx`:

```javascript
const WORKER_URL = 'https://YOUR-NEW-WORKER-URL.workers.dev'
```

## 📋 ข้อมูลที่เก็บ

ข้อมูลที่จะส่งไป Telegram:
- ✅ ชื่อ-นามสกุลเด็ก
- ✅ ช่วงอายุ (6-8 ปี หรือ 9-11 ปี)
- ✅ ชื่อ-นามสกุลผู้ปกครอง
- ✅ เบอร์โทรศัพท์ (10 หลัก)
- ✅ วันที่-เวลาที่สมัคร (โซนเวลาไทย)
- ✅ ข้อมูลคอร์ส (ค่าเรียน, สถานที่)

## 🧪 วิธีทดสอบ

1. รัน development server:
   ```bash
   npm run dev
   ```

2. เปิดเว็บไซต์และไปที่หน้า **Academy**

3. กดปุ่ม **สมัครเรียนตอนนี้**

4. กรอกข้อมูล:
   - ชื่อเด็ก: `น้องทดสอบ`
   - ช่วงอายุ: เลือก `6-8 ปี`
   - ชื่อผู้ปกครอง: `คุณทดสอบ`
   - เบอร์โทร: `0812345678`

5. กดปุ่ม **ส่งใบสมัคร**

6. ตรวจสอบ Telegram ของคุณ (ถ้า Worker ยังใช้งานได้)

## 🔧 การแก้ไขปัญหา

### ปัญหา: ไม่มีข้อความมาใน Telegram

**สาเหตุที่เป็นไปได้:**
1. Telegram Worker URL อาจหมดอายุหรือไม่ทำงาน
2. Bot Token หมดอายุ
3. Chat ID ไม่ถูกต้อง

**วิธีแก้:**
- ตรวจสอบ Console ในเบราว์เซอร์ (F12)
- ดูว่ามี error อะไรใน Network tab
- ลองเข้า URL ของ Worker ดูว่ายังทำงานอยู่ไหม

### ปัญหา: แสดง "เกิดข้อผิดพลาดในการส่งข้อมูล"

**วิธีแก้:**
1. ตรวจสอบ internet connection
2. ตรวจสอบว่า Worker URL ถูกต้อง
3. ดู Console log (`console.warn`) เพื่อดูรายละเอียด error

### ปัญหา: Form validation ไม่ผ่าน

**ตรวจสอบ:**
- ชื่อต้องไม่ว่าง
- เบอร์โทรต้องเป็นตัวเลข 10 หลัก และขึ้นต้นด้วย 0
- ต้องเลือกช่วงอายุ

## 💡 ข้อดีของการใช้ Telegram

✅ **ง่าย** - ไม่ต้องตั้งค่า OAuth หรือ Google Apps Script  
✅ **เรียลไทม์** - ได้รับการแจ้งเตือนทันทีที่มีการสมัคร  
✅ **มือถือ** - ตรวจสอบได้จากมือถือทุกที่  
✅ **ไม่จำกัด** - ไม่มีขอบเขตจำนวนข้อความ  
✅ **ฟรี** - ไม่มีค่าใช้จ่าย  

## 🔐 ความปลอดภัย

- Telegram API มีความปลอดภัยสูง

- ส่งตรงไปยัง Telegram เท่านั้น
